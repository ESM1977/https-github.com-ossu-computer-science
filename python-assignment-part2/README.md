# Restaurant Menu & Order Management System

This project is developed using python to demonstrate how to work with dictionaries, lists, loops, functions, deep copy anf formatted output.  
The program uses the given menu, inventory, and sales data to simulate restaurant operations.

## Project Overview

The project uses 3 provided datasets: 

menu, inventory and sales_log

## Task Included

### Task 1 - Explore the menu : 

The task focusses on reading and analyzing the 'menu' dictionary

- Displays menu items grouped by category
- show item names, price, quantity and availability status
- Computes total no. of items, available items and dispaly expensive item


## Concepts Used

- Lists and Dictionaries  
- Nested data structures  
- Loops and conditional statements

### Task 2 - Cart Operations : 

This task simulates a customer placing an order using a shopping cart by adding item to the cart, checking whether the item is available in the cart or exists in the menu, if existing - updating the quantity, removing an item and updating the qty and cart, generating final order summary.

### Task 3 - Inventory Tracker with Deep Copy:

- Creating a deep copy, Demonstrate that changing the original inventory does not affect the backup, Restoring the original inventory before continuing
- Deduct ordered quantities from inventory based on the final cart, Prevent stock from going below 0, Showing warning if stock is insufficient, Printing reorder alerts for items at or below reorder level
- Printing both:
updated inventory
backup inventory

### Task 4 - Sales Log Analysis
This task analyzes restaurant sales data stored in the sales_log dictionary.
