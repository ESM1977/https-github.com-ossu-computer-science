
# Student Data Processing and Analysis with Python

This project is built using Python. It demonstartes how to clean raw student data, analyse marks, generate class performance grades, and perform string manipulation operations.

## Project Description

This project demonstartes solution for 4 basic Python tasks for understanding the programming fundamentals such as:

- Data Cleaning
- Loops
- Conditional operators
- String Handling
- List operations
- Report formatting

## Task Performed:

### Task 1 - Data Parsing and Cleaning

#### Operations Performed

- Remove leading and trailing spaces from Names
- Convert names to Tilte Case
- Convert Roll No from string to integer
- split marks into list of integers
- validate data
- print student profile card
- display name in uppercase and lowercase for specific rollno

### Task 2 - Marks analysis using control structures

#### Operations Performed

- Print each subject with marks and grade
- Assign grades based on score ranges
- Calculate total and average marks
- Identify highest and lowest scores
- using while loop for new subject entry
- calculate updated average

### Task 3 - Class Performance summary

#### Operations Performed

- Calculated average marks for each student
- Assign Pass/Fail status
- Print a formatted class report
- Calculate class average

### Task 4 - String Manipulation

#### Operations Performed

- Remove extra spaces from start and end
- Convert the essay to title case
- Count the word 'python' in the string
- split the essay into sentances
- replace 'python' with 'python 🐍'
- print and end each statement
